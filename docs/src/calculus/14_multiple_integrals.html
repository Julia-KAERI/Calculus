14.2 다중적분

연습문제

연습문제 14.1 (SNU 14.2.2 (적분과 평균값 정리)) \(n\)-공간의 영역 \(A\) 에서 연속인 함수 \(f\) 가 최대값과 최소값을 가진다고 하자. 이 때 \(A\) 가 곡선연결되어 있으면

\[ \dfrac{1}{\text{vol}(A)} \int_A f=f(P) \]

를 만족하는 점 \(P\in A\) 가 존재함을 보여라.

(해답). \(f\) 의 최대값이 \(P_M\in A\) 에서, 최소값은 \(P_m \in A\) 에서 정해진다고 하자. 그렇다면

\[ f(P_m) = \dfrac{1}{\text{vol}(A)}\int_A f(P_m) \le \dfrac{1}{\text{vol}(A)} \int_A f \le \dfrac{1}{\text{vol}(A)}\int_A f(P_M) = f(P_M) \]

이다. \(A\) 가 곡선연결 되어 있으므로 \(P_M\) 과 \(P_m\) 을 잇는 곡선 \(X(t)\) 을 \(X(0)=P_m,\, X(1) = P_M\) 이 되도록 할 수 있다. \(f\) 가 \(A\) 에서 연속이므로 \(f(X(t))\) 도 연속이며 \(g=f\circ F\) 에 대해 중간값 정리에 의해 문제에서 요구하는 \(P\in A\) 가 존재한다.


연습문제 14.2 (SNU 14.2.3) \(n\) 공간의 연결된 영역 \(A\) 에서 정의된 연속함수 \(f,\,g:A \to \mathbb{R}\) 에 대하여 \(f\) 와 \(g\) 의 평균값을 각각 \(\overline{f},\, \overline{g}\) 로 두면

\[ \overline{f}:\overline{g} = f(P): g(P) \]

가 되는 점 \(P\) 가 \(A\) 속에 존재함을 보여라.

(해답). \(\overline{f},\, \overline{g}\) 는 아래와 같다. \[ \overline f = \dfrac{1}{\text{vol}(A)}\int_A f,\qquad \overline g = \dfrac{1}{\text{vol}(A)}\int_A g. \]
